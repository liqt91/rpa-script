# 工作日志 2026-07-12

## 项目当前状态

### 架构

三层指令体系：`backend`（纯 Python）、`extension`（Python 桩 + JS handler）、`control`（控制流）。

指令定义流：`commands/*.json` → `generate_commands.py` → `src/runtime/commands/{type}_commands/*.py` → `build_extension.py` → `dist/desktop/extension/`。

类型系统：`src/runtime/commands/types/value_types.json`（参数类型 + 值类型）、`categories.json`（分类定义）→ 启动自动 seed DB。

### 开发流程

```bash
# 新增指令
1. 写 commands/xxx.json
2. 写 handler（background_handlers/xxx.js 或 dom_handlers/xxx.js）
3. python scripts/build_extension.py
4. 重启服务器 + 刷新扩展

# 调试
服务器: python -m src.runtime.main (or npm run dev)
前端: src/ui/workflow-editor/ → npm run dev
扩展: chrome://extensions → 加载 dist/desktop/extension/
Service Worker 日志: chrome://extensions → Service Worker 链接
页面 content script 日志: F12 → Console → 选 "content script" 上下文
```
